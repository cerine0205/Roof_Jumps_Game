# Rooftop Jumps Game

Rooftop Jumps is a parkour platformer game set on city rooftops.
The player must jump across buildings, avoid falling, and reach the final destination to win.
The game focuses on precision and smooth movement mechanics.

Project Objective

The goal of this project is to create a simple but enjoyable parkour experience that emphasizes movement control and level completion using Unity.

Features
• Parkour gameplay across rooftops
• Fall = lose / restart
• Reach the end = win
• Smooth player movement and jumping
• Simple level design with clear goal

Gameplay
• Move and jump across rooftops
• Avoid gaps and falling off buildings
• If you fall, you lose and restart
• Reach the final point to win

Technical Overview
• Developed using Unity
• Built with C#
• Collision detection and triggers
• Win/Lose conditions
• Scene management (restart / win screen)

Limitations
• Single level (or limited levels)
• No checkpoints (restart on fall)
• No score system or collectibles
